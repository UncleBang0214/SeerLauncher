# SeerLauncher

项目基于PYQT5，使用Python3.6.8-32bit

主要实现Seer登录器的登录、变速、内置信息、脚本等功能，该登录器主要面向民间服务器版本Seer


部分注意事项：

1.生成QRC文件
引用时写from img import login_logo_rc
pyrcc5 -o login_logo_rc.py img/login_logo.qrc

2.打包
pyinstaller SeerLauncher.spec
